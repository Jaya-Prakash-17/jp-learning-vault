## Bullet Point 1
**"Developed and deployed machine learning models to analyze customer behavior and optimize e-commerce data pipelines using Python."**

- **Actual Work Performed**: As part of my internship/academic work, I built data extraction scripts to clean raw e-commerce datasets, performed exploratory data analysis (EDA), and deployed predictive models to categorize products or predict customer churn.
- **Technical Details**: Python, Pandas (Data Manipulation), Scikit-Learn (Modeling), Matplotlib/Seaborn (Visualization), Jupyter Notebooks, Git.
- **Possible Interview Questions**: 
  - "Walk me through how you handled missing or dirty data in your Yoshops project."
  - "Why did you choose the specific model you used?"
- **Strong Answer**: "The raw dataset had significant missing values in the customer demographic columns. Instead of dropping those rows and losing valuable data, I used median imputation for numerical fields and created a separate 'Unknown' category for categorical fields. I chose a Random Forest classifier because the dataset had high dimensionality and non-linear relationships, which linear regression couldn't capture effectively."
- **Red Flags To Avoid**: Overstating the business impact. Interviewers know this is early-career/internship experience. Be honest that it was a proof-of-concept or academic application.
- **Evidence Supporting Claim**: Jupyter notebooks and Python scripts committed to my personal GitHub repository showing an F1/Accuracy score of over 85%.
